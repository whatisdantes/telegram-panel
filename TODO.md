# UI Design TODO

Цель: привести веб-панель к цельному стилю "Telegram Operator Console": рабочий, быстрый, читаемый интерфейс для управления аккаунтами, чатами, контактами, медиа, приватностью и состояниями аккаунтов.

## 1. Дизайн-направление

- [x] Зафиксировать визуальный стиль: полупрозрачная операторская консоль, не копия Telegram и не декоративный мессенджер.
- [x] Сохранить текущую архитектуру vanilla HTML/CSS/JS без перехода на React.
- [x] Сделать интерфейс одинаково понятным с кастомным фоном и без него.
- [x] Определить 2 темы: темная рабочая и светлая контрастная.
- [x] Убрать случайность в визуале: единые радиусы, тени, отступы, размеры кнопок, цветовые роли.

- [x] Разделить токены для обычного фона и кастомного медиа-фона.
